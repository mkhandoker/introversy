Vocabulary

  • Sprite - A graphics object that can be moved on top of or behind a background.
  • Stage - Where the sprite(s) are going to perform a task
  • Scripts - A set of instructions. In Scratch a script is a single stack of blocks belonging to one sprite or to the stage.
  • Costumes - The different images of a sprite
  • Sounds - Some form of audio
  • Iterate - Repeating a step-by-step procedure several times; also used to refer to single execution of that repetitive procedure
  • Pair Program - Programming with another individual
  • Integrated Development Environment - Provides a developer with a way to create a program, run the program, and debug the program all within one application.
  • Camera Icon - An Icon that allows you to access the camera
  • Allow Flash Player - Giving scratch access to your flash player
  • Save- Saving your progress
  • Import - To transfer previously saved file
  • Select - To choose
  • Delete - To remove
  • Info - Short for information
  • Event - When something happens
  • Boolean Expressions - Evaluates to either true or false; used in the conditional of an if-structure.
  • If then - Something done as a result of something
  • Executes - To run a program or a single instruction.
  • Bug - Part of a program that causes an error or undesired output.
  • Debugging - The process of figuring out why code doesn't behave as expected and eliminating bugs to make it work as desired.
  • CPU Core - The part of the computer that executes one instruction at a time
